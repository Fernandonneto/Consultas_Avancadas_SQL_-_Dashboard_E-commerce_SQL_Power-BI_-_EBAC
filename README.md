# 🧠 Consultas Avançadas SQL: Dashboard de E-commerce com SQL e Power BI

Este projeto faz parte do Módulo Operações Avançadas SQL do curso de Cientista de Dados da EBAC e tem como objetivo aplicar técnicas de SQL, tratamento de dados e visualização no Power BI para análise de desempenho de um e-commerce.

A atividade foi desenvolvida a partir da integração entre bases de transações e clientes, permitindo explorar métricas de vendas, comportamento de clientes assíduos e padrões de consumo.

## 🧩 Etapas Desenvolvidas

**1. Integração das bases com SQL:** \
Utilização de LEFT JOIN para preservar todas as transações da tabela principal e complementar os dados com informações de clientes.

**2. Tratamento dos dados no Power Query:** 
- Remoção de colunas desnecessárias;
- Padronização dos nomes das colunas;
- Tratamento de valores ausentes com a categoria “Não informado”.

**3. Criação de métricas e KPIs:** 
- Receita total de vendas;
- Quantidade de clientes;
- Clientes assíduos (clientes com 2 ou mais compras);
- Clientes não assíduos.

**4. Construção do dashboard no Power BI:** \
Desenvolvimento de visualizações interativas voltadas à análise comercial e de comportamento dos consumidores.

## 📊 Visualizações e Análises

- Valor total de vendas
- Quantidade de clientes assíduos
- Categorias mais vendidas
- Valor médio por categoria
- Produtos mais vendidos para clientes assíduos
- Características dos clientes assíduos
    - Gênero
    - Profissão
    - Estado
- Características dos clientes não assíduos
    - Gênero
    - Profissão
    - Estado

## 📈 Resultados e Insights

- Identificação das categorias e produtos com maior volume de vendas.

- Análise do comportamento de clientes recorrentes e não recorrentes.

- Comparação de perfis demográficos entre diferentes grupos de clientes.

- Criação de métricas estratégicas para acompanhamento do desempenho comercial.

## ✅ Conclusão

O projeto demonstra a integração entre SQL e Power BI para construção de dashboards analíticos e geração de insights estratégicos. A combinação entre tratamento de dados, modelagem e visualização permitiu analisar o desempenho do e-commerce de forma clara e interativa, apoiando a tomada de decisão baseada em dados.

A partir do dashboard desenvolvido, é possível identificar as categorias com melhor e pior desempenho em vendas, compreender o perfil dos clientes de cada segmento e analisar quais categorias contribuem para a fidelização dos consumidores. Essas análises possibilitam a criação de estratégias mais assertivas para captação e retenção de clientes, otimização de campanhas comerciais e definição dos nichos de mercado com maior potencial de crescimento.
